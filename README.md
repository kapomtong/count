# Drama Court ⚖️

AI-Powered Drama Judgment System

## Deploy to Vercel

1. Push to GitHub
2. Import to Vercel (vercel.com/new)
3. Add Environment Variable:
   - `ANTHROPIC_API_KEY` = your key
4. Deploy!

## Features
- Claude AI judgment (Opus/Sonnet/Haiku)
- Firebase Realtime Database
- File upload (video/audio/image)
- Voice recording
- Member roles (Party A/B/Witness)
- Public voting
- Mobile-first dark UI
